# Setka CMS

Библиотека ядра Setka CMS на базе Yii2.

## Установка

```bash
composer install
```

## Использование

Добавьте пакет в свой проект и подключите необходимые компоненты CMS.
Подробная документация доступна в репозитории.

## Тесты

```bash
composer test
```

## Анализ кода

```bash
composer cs -- --dry-run
composer stan
composer rector -- --dry-run
```
